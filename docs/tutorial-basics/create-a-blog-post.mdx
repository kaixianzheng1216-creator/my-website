# 创建博客文章

Docusaurus 会为每篇博客文章创建一个 **页面**，同时还会生成一个 **博客索引页**、**标签系统**、**RSS** 订阅等。

## 创建你的第一篇文章

在 `blog/2021-02-28-greetings.md` 创建一个文件：

```md title="blog/2021-02-28-greetings.md"
---
slug: greetings
title: 问候！
authors:
  - name: Joel Marcey
    title: Docusaurus 1 联合创建者
    url: https://github.com/JoelMarcey
    image_url: https://github.com/JoelMarcey.png
  - name: Sébastien Lorber
    title: Docusaurus 维护者
    url: https://sebastienlorber.com
    image_url: https://github.com/slorber.png
tags: [greetings]
---

恭喜，你已经创建了第一篇文章！

你可以随意尝试并按自己的想法编辑这篇文章。
```

一篇新的博客文章现在可以在 [http://localhost:3000/blog/greetings](http://localhost:3000/blog/greetings) 访问。

# Markdown 功能

Docusaurus 支持 **[Markdown](https://daringfireball.net/projects/markdown/syntax)**，并提供一些 **额外的功能**。

## Front Matter

Markdown 文档的顶部有一段元数据，称为 [Front Matter](https://jekyllrb.com/docs/front-matter/)：

```text title="my-doc.md"
// highlight-start
---
id: my-doc-id
title: My document title
description: My document description
slug: /my-custom-url
---
// highlight-end

Markdown content
```

## 标题 {/* #my-heading-id */}

Docusaurus 支持使用标准的 `#` 语法编写 Markdown 标题，这些标题会自动加入目录。`#` 的数量对应标题的级别。

```md
## Headings

My text
```

### 自定义标题 ID {/* #my-custom-id */}

在标题文本后加上 `{/* #my-custom-id */}`，可以为标题显式指定一个锚点 ID，用于链接跳转。

```md
### Heading Ids {/_ #my-custom-id _/}
```

## 链接

支持常规的 Markdown 链接，可以使用 URL 路径或相对文件路径。

```md
Let's see how to [Create a page](/create-a-page).
```

```md
Let's see how to [Create a page](./create-a-page.mdx).
```

**效果：** Let's see how to [Create a page](./create-a-page.mdx).

## 图片

支持常规的 Markdown 图片。

你可以使用绝对路径来引用 `static` 目录中的图片（`static/img/docusaurus.png`）：

```md
![Docusaurus logo](/img/docusaurus.png)
```

![Docusaurus logo](/img/docusaurus.png)

你也可以使用相对于当前文件的路径来引用图片。这对于将图片与使用它的 Markdown 文件放在同一目录下非常有用：

```md
![Docusaurus logo](./img/docusaurus.png)
```

## 代码块

支持带语法高亮的 Markdown 代码块。

````md
```jsx title="src/components/HelloDocusaurus.js"
function HelloDocusaurus() {
  return <h1>Hello, Docusaurus!</h1>;
}
```
````

```jsx title="src/components/HelloDocusaurus.js"
function HelloDocusaurus() {
  return <h1>Hello, Docusaurus!</h1>;
}
```

## 提示框

Docusaurus 提供了一种特殊语法来创建提示与警告框：

```md
:::tip[My tip]

Use this awesome feature option

:::

:::danger[Take care]

This action is dangerous

:::
```

:::tip[小贴士]

使用这个超棒的功能选项

:::

:::danger[请注意]

此操作存在风险

:::

## MDX 与 React 组件

[MDX](https://mdxjs.com/) 可以让文档更加 **交互式**，并允许在 Markdown 中使用任何 **React 组件**：

```jsx
export const Highlight = ({children, color}) => (
  <span
    style={{
      backgroundColor: color,
      borderRadius: '20px',
      color: '#fff',
      padding: '10px',
      cursor: 'pointer',
    }}
    onClick={() => {
      alert(`You clicked the color ${color} with label ${children}`)
    }}>
    {children}
  </span>
);

This is <Highlight color="#25c2a0">Docusaurus green</Highlight> !

This is <Highlight color="#1877F2">Facebook blue</Highlight> !
```

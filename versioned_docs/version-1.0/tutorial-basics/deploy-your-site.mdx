# 部署你的站点

Docusaurus 是一个 **静态站点生成器**（也称为 **[Jamstack](https://jamstack.org/)**）。

它会将你的站点构建为简单的 **静态 HTML、JavaScript 和 CSS 文件**。

## 构建站点

为 **生产环境** 构建站点：

```bash
npm run build
```

静态文件会生成在 `build` 目录中。

## 部署站点

在本地测试生产构建：

```bash
npm run serve
```

`build` 目录现在可以通过 [http://localhost:3000/](http://localhost:3000/) 进行访问。

现在，你可以将 `build` 目录 **几乎部署到任何地方**，**免费** 或以极低的成本完成（请阅读 **[部署指南](https://docusaurus.io/docs/deployment)**）。

# 教程简介

让我们在 **5 分钟内认识 Docusaurus**。

## 快速开始

通过 **创建一个新站点** 来开始。

或者使用 **[docusaurus.new](https://docusaurus.new)** **立即体验 Docusaurus**。

### 环境要求

- [Node.js](https://nodejs.org/en/download/) 20.0 或更高版本：
  - 安装 Node.js 时，建议勾选所有与依赖项相关的复选框。

## 生成新站点

使用 **经典模板** 生成一个新的 Docusaurus 站点。

运行以下命令后，经典模板会自动添加到你的项目中：

```bash
npm init docusaurus@latest my-website classic
```

你可以在命令提示符、PowerShell、终端或代码编辑器的任何集成终端中输入此命令。

该命令还会安装运行 Docusaurus 所需的所有依赖。

## 启动站点

运行开发服务器：

```bash
cd my-website
npm run start
```

`cd` 命令会切换当前工作目录。要使用你新创建的 Docusaurus 站点，需要在终端中先进入该目录。

`npm run start` 命令会在本地构建你的网站，并通过开发服务器提供服务，访问 http://localhost:3000/ 即可查看。

打开 `docs/intro.md`（即本页）并修改其中几行：站点会 **自动重新加载** 并显示你的更改。
